### How to Run

To run this project locally:

1. **Clone the repository:**
    ```bash
    git clone https://github.com/your-repo/flutter-login-ui.git
    ```

2. **Navigate to the project directory:**
    ```bash
    cd flutter-login-ui
    ```

3. **Install dependencies:**
    ```bash
    flutter pub get
    ```

4. **Run the project:**
    ```bash
    flutter run
    ```
